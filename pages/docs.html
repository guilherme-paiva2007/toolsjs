Adicionar páginas

Páginas podem ser adicionadas dinamicamente de duas maneiras.

Carregando um diretório:

method ServerManager.openPageDir(dirpath: string, pathbase: string): void

Onde dirpath é a localização do diretório que será lido, e pathbase é o caminho relativo base para ser inserido antes do nome dos arquivos do diretório.

Carregando uma lista:

method ServerManager.openPageList(array: PageListObject[], path: string): void

Onde array é a lista de objetos que representam páginas, e path é o caminho de origem onde os arquivos da lista serão procurados.

Abrir WebSocket simultaneamente

method ServerManager.openWebSocket(path?: string): void

Abre uma conexão WebSocket compartilhando o mesmo hostname e porta do servidor.
